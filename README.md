Elektrofon PDB
==============

### Installation

```bash
$ yarn install
```

### Start scanner app

```bash
$ yarn start
```

### Generate serials

```bash
$ node src/generate-serials.js [identifier [count]]
```

### Generate QRcodes

```bash
$ node src/generate-qrcodes.js
```

### Generate PDB site

```bash
$ node src/generate-site.js
```

### Deploying PDB site

```bash
$ yarn run deploy
```
